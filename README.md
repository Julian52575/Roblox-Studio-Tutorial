# Roblox Studio Tutorial

Find here my tutorial on using Roblox Studio after my year working on it at Keimyung University.  


## 1. Familiarizing with Roblox Studio

First, download studio [here](https://create.roblox.com/) and create a Roblox account if you don't have one yet.  

Click on <img width="132" height="44" alt="+ New Experience" src="https://github.com/user-attachments/assets/2bb3345d-d62a-444f-a3de-87e2f21bef82" /> to start working.  

> [!TIP]
> Feel free to close all sub-windows. We will reopen them as we need ;)

### Game view

<img width="640" height="382" alt="image" src="https://github.com/user-attachments/assets/846b4322-0969-47f0-ac40-e672912ca2dc" />  

You can move the camera using `w` `a` `s` `d`.  
`q` and `e` will move the camera up and down.  
`i` and `o` will zoom in or zoom out.  
`Right click` will select an object.  
Pressing `Left click` and moving the mouse will rotate the camera.  
Using `f` with a selected object will focus the camera on it.  

### Explorer

<img width="172" height="319" alt="image" src="https://github.com/user-attachments/assets/b5aa4403-857a-4c7b-b822-77fac65e553e" />  

There are many things to see inside the Explorer.  
Developers manly use these 4 folders:
| Service | Usage |
| :------------------- | :----------: |
| Workspace            | Game objects     |
| ReplicatedStorage              | ModuleScripts shared by server and client |
| ServerScriptService               | ModuleScripts and Scripts used by the server |
| StarterPlayer               | LocalScripts attach to each player |

> [!NOTE]
> Don't worry about ModuleScripts for now

### Output

A terminal like display of your and Studio's scripts.

### AI Assistant

You can open the built-in AI Assistant using a button on the top right corner.  
It will update the game's objects and script in response of your prompts.  

> [!CAUTION]
> AI has complete control over your projet and will update or delete anything it wants without warning
